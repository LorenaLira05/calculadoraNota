# Boletim Escolar

Aplicação web simples feita com **HTML, CSS e JavaScript** para simular um boletim escolar de forma prática.

## Funcionalidades

- Entrada das 4 notas principais do aluno
- Cálculo da média inicial
- Verificação de status: **aprovado** ou **recuperação**
- Uso da nota de recuperação para substituir a menor nota (quando for maior)
- Verificação de etapa final com nota final, se necessário
- Exibição de um resumo completo ao clicar em **Resumo**

## Como executar

1. Abra o arquivo `index.html` no navegador.
2. Preencha as notas do aluno.
3. Clique no botão **Resumo** para ver o resultado escolar.

## Estrutura do projeto

- `index.html` - estrutura da interface
- `style.css` - estilos da página
- `script.js` - regras de cálculo e geração do resumo

## Observação

As notas devem estar no intervalo de **0 a 10**.
